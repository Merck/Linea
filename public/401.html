Kerberos setup needed

Looks like you are running Chrome or Firefox on Mac OS X. To configure your browser for use with Linea, please follow the steps below.

Google Chrome

Close Google Chrome and in Terminal, run the following commands:

$ defaults write com.google.Chrome AuthServerWhitelist "*.Linea.com"

$ defaults write com.google.Chrome AuthNegotiateDelegateWhitelist "*.Linea.com"

Firefox

Close Firefox and in Terminal, run the following commands:

$ defaults write com.google.Chrome AuthServerWhitelist "*.Linea.com"

$ defaults write com.google.Chrome AuthNegotiateDelegateWhitelist "*.Linea.com"
